
  FltCntrl представляет собой компонент, предназначеный для организации визуальных
элементов для задания фильтров в интерфейсах. Как это работает можно увидеть вызвав команду
"Спецоснастка\Отчеты\Отчет о наличии" и попробовав задать фильтр.
  Примеры использования в коде можно найти в каталоге VIP\MBP\ в файлах RepParam.vip,
REGLGETP.VIP, REGLSPAR.VIP и UTILS\MBP2OS.VIP. Причем интерфейс в файле REGLGETP.VIP
почти не не содержит лишнего кода.
  Как правило, данный компонент используется в интерфейсах запрос параметров для
построения отчета и т.д. Для последующей выборки заданных в фильтре критериев можно
использовать интерфейс fltBlock.
  Для сокращения затарат на подсчет количеcтва помеченных записей можно использовать
функцию FilterBlock.ExtrCntFromName. Пример смотри в вышеперечисленных исходниках.
